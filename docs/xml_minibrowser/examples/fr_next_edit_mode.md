## next_edit_mode fragment 

!!! Demo
    [Try on your phone](xml/fr_next_edit_mode.xml)

Change the edit mode on *InputField*.
Just select the first menu and see the value from *edit_mode* and *next_edit_mode* callback and also the input mode (lower-case, upper-case and number) change when pressing *F1*.
Don't forget to add the IP address or hostname as minibrowser trusted host, when running this example
