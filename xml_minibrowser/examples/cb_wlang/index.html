wlang callback #

Demo

Try on your phone

Returns the localized web string for a given language key.

Some valid key and the result from two different WUI language. The result may change if you use customized language or there is an update to language file. To change WUI language go to http://phone_ip/prefs.htm

<?xml version="1.0" encoding="UTF-8"?>
<SnomIPPhoneMenu xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:noNamespaceSchemaLocation="../minibrowser_test.xsd">
    <!-- Deutsch --><!-- English -->
    <!-- DE --><MenuItem name="$(wlang:wlang_iso_code)"></MenuItem><!-- EN -->
    <!-- Deutsch --><MenuItem name="$(wlang:wlang_web_language_name)"></MenuItem><!-- English -->
    <!-- Keine --><MenuItem name="$(wlang:wlang_adr_ctnone)"></MenuItem><!-- None -->
</SnomIPPhoneMenu>
